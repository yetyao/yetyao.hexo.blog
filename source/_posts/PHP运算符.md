## PHP运算符


```
<?php
$a = 0;
$b = 0;
if($a=3||$b=3){
	$a++;
	$b++;
}
echo $a.'-'.$b;
//1-1
```
> || *优先级*高于 =
> `if($a=3||$b=3)` 相当于 `if($a=(3||$b=3))`
> `$a = (3||$b=3)` 又因为*短路*导致 `$a=true`
> `if($a=true)` 相当于 `if(true)` 此时因为*意外赋值* $a为true,$b为0
> `$a++`和`$b++` 进行*后加加* 
> 故$a为1 $b为1 结果为 1-1

```
<?php
$a = 0;
$b = 0;
if($a=3||$b=3){
    $a;
    $b++;
}
echo $a.'-'.$b;
//1-1
```
> || *优先级*高于 =
> `if($a=3||$b=3)` 相当于 `if($a=(3||$b=3))`
> `$a = (3||$b=3)` 又因为*短路*导致 `$a=true`
> `if($a=true)` 相当于 `if(true)` 此时因为*意外赋值* $a为true,$b为0
> `$b++` 进行*后加加*  `echo true;` 为1
> 故$a为true $b为1  结果为 1-1 

```
$a = 0;
$b = 0;
if($a=3 or $b=3){
    $a++;
    $b++;
}
echo $a.'-'.$b;
//4-1
```
> or *优先级*低于 =
> 因为*短路* `if($a=3 or $b=3)` 相当于 `if($a=3)`
> `if($a=3)` 相当于 `if(3)` 此时因为*意外赋值* $a为3,$b为0
> `$a++`和`$b++` 进行*后加加* 
> 故$a为4 $b为1 结果为 4-1
